> **AVANT DE COMMENCER**
> 
> Pour utiliser un format personnalisé, vous devez avant tout créer un dossier `YourDatabase.4dbase/Resources/Mobile/formatters`.
> 
> Glissez ensuite le dossier du format dans le dossier que vous venez de créer.
